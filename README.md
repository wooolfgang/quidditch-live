# Quidditch League

A real-time app for Quidditch matches. http://quidditch-league.herokuapp.com/
Side-project I did to practice on Redux and realtime implementations using Socket.io

## Getting Started

npm run install
npm run seed:all
npm run start:dev
npm run bundle
